## 使用说明

### 自定义不死图腾皮肤

不死图腾皮肤存放路径：`assets/minecraft/textures/item/doll`

将目录下的 `steve.png` 替换为你自己的皮肤文件，即可完成自定义。

## 适配与兼容

- 本版本为基础适配版本，重点在于兼容声明与整合稳定性。
- 推荐搭配 OptiFine 或 钠（Sodium）+ Continuity 使用，以获得更完整的视觉效果。
- 草方块已默认切换为更标准的原版模型实现（适配 1.21.1 / BetterGrassify 2.6.1 及后续 26.1.x）。
- 若安装 BetterGrassify，建议保留其资源包兼容模式开启，由模组侧执行 Better Grass 逻辑。
- 旧版“全草面原版可用”方案已保留为模型备份：assets/minecraft/models/block/grass_block_legacy_full.json。
